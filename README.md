## 📌 About

**Smart AI Resume Analyzer** is an AI-powered tool that helps you analyze, optimize, and craft professional resumes that stand out to recruiters.

> 🎯 **Goal:** Help job seekers land their dream roles with data-driven resume feedback.

**Submitted for partial fulfilment of degree at SRM University, Amaravati**

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📄 **Resume Parsing** | Extract text from PDF/DOCX resumes |
| 🧠 **AI Analysis** | NLP-powered content evaluation |
| 📊 **Smart Scoring** | Get a resume score with tips |
| 💼 **Job Matching** | Match resume with job descriptions |
| 🎨 **Modern UI** | Clean Streamlit interface |
| 📥 **Export** | Download PDF reports |

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/YAMINI-IKI/smart-ai-resume-analyzer.git
cd smart-ai-resume-analyzer

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
