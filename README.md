## Setup Instructions

1. Clone the repository:
   git clone https://github.com/YAMINI-IKI/smart-ai-resume-analyzer.git
   cd smart-ai-resume-analyzer

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py
